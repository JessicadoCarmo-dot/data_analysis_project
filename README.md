# Análise de Tweets sobre a Covid-19:
Projeto de Big Data com foco na análise de tweets relacionados à Covid-19,utilizando dados extraídos do Kaggle. 
O objetivo é identificar países com maior volume de discussões,sentimentos predominantes, meios de acesso à plataforma
e os termos mais utilizados pelos usuários.

# Fonte dos Dados:
Dataset público disponibilizado no Kaggle, de autoria de Gabriel Pedra, contendo tweets relacionados 
à pandemia da Covid-19.

# Tecnologias Utilizadas:
- Python
- PySpark
- Pandas
- Spark
- HTML, CSS
- Plotly
- Visual Studio Code

# Análises Realizadas:
- Países com discussões mais ativas  
- Análise de sentimentos (positivo, negativo e neutro)  
- Meios de acesso ao Twitter (web ou mobile)  
- Contagem das palavras mais frequentes nos tweets

# Visualização:
Dashboard interativo desenvolvido com Plotly, HTML e CSS, apresentando 
gráficos de barras e pizza para facilitar a interpretação dos resultados.

# Possíveis Melhorias:
- Integração de dados em tempo real
- Aplicação de técnicas de Inteligência Artificial
- Expansão para outros temas além da pandemia
